# Direct-Cross-Graph Baselines Guide

This directory contains algorithm-specific direct cross-graph baseline outputs.

Purpose:

- Preserve baseline comparisons for each algorithm in a dedicated subtree.
- Keep raw, evaluated, and factorial baseline artifacts grouped together.

Working rules:

- Do not rename these files unless the baseline contract changes.
- Keep the `algo1`, `algo2`, and `algo3` subtrees aligned in structure.
- Add deeper guides only if a subtree introduces a new model or output family.
