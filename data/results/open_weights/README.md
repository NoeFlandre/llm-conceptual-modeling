# Open Weights Results

This bucket holds the paper-facing open-weight study outputs.

- `hf-paper-batch-canonical/` contains the canonical Qwen/Mistral study outputs, the finished-run ledger, and the variance-decomposition bundle.

The canonical results tree is the source of truth for the paper-facing open-weight runs.
