# Results Archives

This tree holds preserved operational artifacts that are no longer part of the top-level paper workflow.

- `olmo/` contains archived OLMO runs and smoke artifacts.
- `operational/` contains archived `*-current` workdirs and canonical batch runtime leftovers.
- `stale-shards/` contains archived shard trees and other drained batch layouts.

The archive is intentionally preserved for provenance, but it should stay out of the way of the canonical paper outputs in `results/open_weights/hf-paper-batch-canonical/`.
