# Baseline Comparison Audit Artifacts

These files support the non-LLM baseline findings recorded in `paper/revision-tracker.md`.

## Baseline Definition

The baseline is the deterministic `direct-cross-graph` heuristic generated by `lcm baseline ...`.

For Algorithms 1 and 2, it proposes direct edges from the mother graph whose endpoints fall across the requested two subgraphs. For Algorithm 3, it reuses the same structural idea inside the Algorithm 3 raw-output shape and then runs the existing evaluation pipeline.

This is a structural heuristic, not a learned model. It is useful as an offline comparator because it is deterministic, auditable, and evaluated with the same downstream metric code as the imported LLM outputs.

## Source Data

- Imported evaluated model outputs under `data/results/*/*/evaluated/`
- Deterministic baseline evaluated outputs under `data/baselines/direct-cross-graph/*/evaluated/`

## Files

- `algo1_model_vs_baseline.csv`
  Mean metric comparison for each imported ALGO1 model against the ALGO1 direct-cross-graph baseline.
- `algo2_model_vs_baseline.csv`
  Mean metric comparison for each imported ALGO2 model against the ALGO2 direct-cross-graph baseline.
- `algo3_model_vs_baseline.csv`
  Mean recall comparison for each imported ALGO3 model against the ALGO3 direct-cross-graph baseline.
- `all_models_vs_baseline.csv`
  Combined per-model comparison table across all algorithms and metrics.
- `baseline_advantage_summary.csv`
  Aggregated count and magnitude summary showing whether any imported model beats the deterministic baseline.

## Interpretation Notes

These artifacts compare evaluated means, not provider internals. They show how the imported model outputs compare to one deterministic structural heuristic on the same downstream metrics.

The baseline uses mother-graph structure directly. It should therefore be interpreted as a conservative graph heuristic that exposes how much signal is recoverable from the available structure alone, not as a claim that live LLM generation is unnecessary in every research setting.
